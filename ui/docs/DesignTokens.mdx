# Design Tokens

DesignReady.ai uses a two-layer token system: Figma-native variables at runtime, with CSS fallbacks for Storybook. All tokens are synced to Figma as Variables in the **DesignReadyAi** collection (38 variables).

## Dark Theme Only

DesignReady.ai is a Figma plugin — it always runs in Figma's dark theme. There is no light mode.
