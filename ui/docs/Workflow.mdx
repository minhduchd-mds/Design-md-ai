# Recommended Workflow

## Single Component Scan

1. Select a component in Figma
2. Open DesignReady.ai plugin
3. Click "Scan Component"
4. Review score — fix issues if below 75
5. Copy prompt — paste into Claude
6. Approve Skill Sync when prompted
7. Get production-ready code

## Batch Scan (Multiple Components)

1. Select 5–10 components in Figma (mix of atoms + molecules)
2. Click "Batch Scan"
3. Review atomic build order (atoms, molecules, organisms)
4. Copy batch prompt — paste into Claude
5. Claude builds in correct dependency order
6. Approve Skill Sync updates

## Setting Up a Profile

1. Go to Setup tab
2. Click "+ New Profile"
3. Click "Import from Figma" to auto-import tokens
4. Add tech stack (e.g. "React + TypeScript + Tailwind")
5. Add guidelines ("BEM naming", "8px grid")
6. Save profile

## Performance Tips

| Selection | Nodes | Speed |
|-----------|:-----:|-------|
| Single atom (Button) | under 100 | Instant |
| Molecule (ProductCard) | 100–500 | Fast, under 2s |
| Organism (Header) | 500–1000 | Slower, 5–10s |
| Full page | 1000+ | Not recommended |

- Scan organisms individually, never entire pages
- Use Batch Scan to process multiple small components
- If a scan takes more than 5s, select a smaller subtree

## Token Cost Awareness

| Scenario | Input Tokens | Sonnet Cost |
|----------|:-----------:|:-----------:|
| Single component | 800–2,000 | ~$0.01 |
| Batch (8 components) | 10,000–18,000 | ~$0.10–0.30 |
| Batch + corrections | 20,000–36,000 | ~$0.20–0.60 |

Check the token estimate next to the Copy button before pasting.
